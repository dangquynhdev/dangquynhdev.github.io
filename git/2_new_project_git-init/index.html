git init

được sử dưng để tạo, khởi tạo một kho chứa Git mới (Git Repo) ở local (local repo)
trong folder ẩn .git có file tên HEAD cũng được tạo ra - nó trỏ đến commit hiện tại.
Sau lệnh này bạn có một Repo ở local và bắt đầu thực thi được các lệnh khác của Git.

git init --bare

tạo một repo tại server

git clone

Bạn chú ý còn có lệnh git clone nó cũng giúp cho bạn có được một Repo mới, tuy nhiên git clone có điểm khác biệt là nó tạo ra một bản sao của một Repo khác

Lệnh git init --bare

tạo ra một Repo Git mà nó chỉ có chức năng lưu trữ - không có thư mục làm việc
Loại dự án Git này thì bạn có thể truy cập, lưu trữ, nhưng không soạn thảo, sửa file, thực hiện commit trực tiếp tại dự án
Thường tạo loại dự án này để lưu trữ như là Remote Repo (Tạo Repo git trên Server), từ đó lấy về Local (lệnh git clone), và để local đẩy dữ liệu Git lên
