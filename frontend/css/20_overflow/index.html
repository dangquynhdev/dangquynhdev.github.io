overflow

The overflow property only works for block elements with a specified height.
In OS X Lion (on Mac), scrollbars are hidden by default and only shown when being used (even though "overflow:scroll" is set).

value

visible - Default. The overflow is not clipped. The content renders outside the element's box
hidden - The overflow is clipped, and the rest of the content will be invisible
scroll - The overflow is clipped, and a scrollbar is added to see the rest of the content
auto - Similar to scroll, but it adds scrollbars only when necessary

overflow-z and overflow-y

khi một trục set giá trị khác với visible thì trục còn lại cũng chỉ có thể nhận giá trị scrol, auto hoặc hidden chứ ko thể là visible
