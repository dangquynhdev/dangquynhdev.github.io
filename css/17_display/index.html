display default of script element

The script element uses display: none; as default.

display none vs visible hidden

none: The element will be hidden, and the page will be displayed as if the element is not there
hidden: The element will be hidden, but still affect the layout

display inline

Setting the display property of an element only changes how the element is displayed, NOT what kind of element it is. So, an inline element with display: block; is not allowed to have other block elements inside it.

inline vs inline-block

inline không set được width, height inline phần lề/ đệm trên dưới không được tôn trọng

inline-block vs block

inline-block không thêm một ngắt dòng sau phần tử
