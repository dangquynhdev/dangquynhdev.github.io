responsive table

A responsive table will display a horizontal scroll bar if the screen is too small to display the full content set overflow:auto tại thẻ bao của table, table: width 100%

In OS X Lion (on Mac), scrollbars are hidden by default and only shown when being used (even though "overflow:scroll" is set).

The <colgroup> element is used to style specific columns of a table.
