[attribute] [attribute="value"]
[attribute~="value"] selector is used to select elements with an attribute value containing a specified word
[attribute|="value"] is used to select elements with the specified attribute, whose value begins with value, or the specified value followed by a hyphen (-). (mustbe whole word)
[attribute^="value"] whose value starts with the specified value
[attribute$="value"] whose attribute value ends with a specified value
[attribute*="value"] whose attribute value contains a specified value
